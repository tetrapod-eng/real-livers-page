Q&A

よくある質問

Q
初期費用はかかりますか？
add
初期費用や事務所の登録料や解除料等はすべて無料です。
Q
配信ノルマはありますか？
add
ノルマはございません。自宅で配信することができるので夜寝る前のすきま時間や休みの空いた時間に、気軽に配信して頂けます。
Q
顔出し無しでもできますか？
add
はい、可能です。弊社は、顔出し無しでの配信が可能なアプリもご案内できます。
Q
未経験でもできますか？
add
はい、可能です。配信未経験からスタートしたライバーも多く在籍しており、マニュアルやノウハウがあるため安心して始めて頂けます。
